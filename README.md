# compsafe\_2025

This is a repository for the codes used for the paper submitted in COMPSAFE 2025.



Program files:

* all\_season, data

&nbsp;	Data taken from Japan Meteorological Agency (https://www.data.jma.go.jp/risk/obsdl/index.php#)

* huffman.py

&nbsp;	Huffman tree creation methods

* send\_if\_flood.py

&nbsp;	Flood Probability calculation methods
